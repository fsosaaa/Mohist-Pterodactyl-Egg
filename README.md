# Mohist Multi-Version Egg for Pterodactyl Panel

An enhanced and complete egg for Pterodactyl Panel that supports multiple versions of Mohist Server, combining the best of Minecraft Forge with Bukkit/Spigot plugin support.

## Key Features

- **Multiple Supported Versions**: 1.12.2, 1.16.5, 1.18.2, 1.19.2, 1.19.4, 1.20.1, 1.20.2, 1.21.1, 1.21.4
- **Automatic Download**: Automatic download from multiple sources with fallback
- **Robust Error Handling**: Complete download verification and configuration
- **Performance Optimization**: Pre-configured JVM arguments for better performance
- **Dual Compatibility**: Simultaneous support for Forge mods and Bukkit/Spigot plugins
- **Multi-Java Support**: Support for Java 8, 11, 17, and 21 based on version
- **Intelligent Configuration**: Automatic configuration based on Minecraft version
